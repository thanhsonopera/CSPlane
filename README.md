# CSPlane

Với mục đích đem đến cho các bạn một phương pháp giải trí hiệu quả sau
những giờ học và làm việc căng thẳng, mình xin giới thiệu sản
phẩm **CS Plane**.

------------------------------------------------------------------------

## Các chức năng

### Play

Phần chơi chính gồm các vật thể như: - Máy bay đồng minh - Máy bay
địch - Đạn - Tên lửa - Bảng tính điểm - Bộ đếm thời gian

Độ khó sẽ tăng dần theo thời gian, mang lại trải nghiệm thử thách và hấp
dẫn hơn.

### Leaderboard

Bảng xếp hạng dựa trên: - Thành tích điểm số - Thời gian sống sót

### Setting

-   Thay đổi nền chơi trong chế độ Play\
-   Thay đổi nhạc nền

### Tutorial

Hướng dẫn cách chơi chi tiết dành cho người mới bắt đầu.

### Exit

Thoát khỏi trò chơi.

------------------------------------------------------------------------

## Hướng dẫn cài đặt & chạy chương trình

``` bash
mingw32-make compile
.\run.bat
```

------------------------------------------------------------------------

## Yêu cầu hệ thống

-   Windows OS
-   MinGW (mingw32-make)
-   Command Prompt hoặc Terminal hỗ trợ chạy file `.bat`

------------------------------------------------------------------------

## Mục đích phát triển

CSPlane được phát triển nhằm: - Giải trí sau giờ học và làm việc căng
thẳng\
- Rèn luyện phản xạ và khả năng xử lý tình huống\
- Tạo môi trường cạnh tranh lành mạnh thông qua bảng xếp hạng
